Plotting Code Generation Benchmarks

为 F-A GroundedChart 实验选型 — 专注数据图表代码生成方向的 benchmark 汇总

Benchmark 卡片
F-A 实验选型建议

主实验（3 个 bench，覆盖面广）：

1. SciGenBench (1.4k) — 最大的科学图表专用 bench，5 个学科，AAAI 2026

2. VisEval (2.5k) — NL2Vis 社区最权威 bench，微软出品，IEEE VIS 2024

3. PlotCraft (1k) — 48 种图表类型，阿里 Qwen 团队

泛化验证（2 个 bench）：

4. PaperBananaBench (500) — ACL 2026，有 code exec rate 指标

5. MatPlotBench (100) — 经典小 bench，适合 ablation / case study

关键策略：这些 bench 原本不测"数据保真度"——你需要用 NumericTrace 的 AST 方法在这些 bench 上额外测量编造率。这反而是加分项：揭示现有 bench 忽略的盲区。
