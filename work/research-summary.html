Executive Summary

Page 01 / 07 — F-A GroundedChart Research Report 2026-04-19

Core Thesis

现有图表生成系统都在优化"图好不好看"，没有人管"数据对不对"

Key Finding: VisEval 63.2% failed, 953 data_check failures; Text2Vis 91.7% failures from answer mismatch; 6 competitors none solve data fidelity

Recommended Reframing

F-A 最好被定义为：
an execution-grounded verification-and-repair framework for faithful language-to-chart generation.

  • 任务边界：输入是自然语言查询 + 结构化数据 + 初始图表代码；目标是在渲染前验证并修复数据处理逻辑
  • 核心对象：aggregation / filter / group-by / join / sort / type mismatch，而不是泛化的"科学图表真实性"
  • 论文定位：强调 NL 意图、结构化数据操作、以及最终绘图数据之间的语义对齐，这比单纯做一个 data check agent 更像 EMNLP 论文

Key Takeaways
  • VisEval 的 data_check 是最大失败源（953例），60% 根因是 missing_groupby
  • Text2Vis 的 answer_mismatch 占 91.7%，51% 是 wrong_entity，33% 是 wrong_value
  • 6 个竞品系统（CoDA / VisPath / PlotGen / MatPlotAgent / METAL / RAGvis）无一专门解决数据保真度
  • RAGvis (Google, EMNLP 2025) 实测 5/5 代码执行失败，确认其为 EDA 推荐器而非 NL→chart
  • 因此最稳的论文版本不是广义 F-A，而是面向 language-to-chart generation 的 verification-and-repair framework
